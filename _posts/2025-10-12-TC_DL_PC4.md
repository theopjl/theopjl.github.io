- the **Testing set** : 15% of the points

### 2. Splitting the data into training/validation/testing datasets using random generated indices

*- Splitting in different sizes*

*- Generating random indices used for splitting the datasets*

*- Building the datasets using the random generated indexes*

```python
from torch.utils.data import TensorDataset, random_split, DataLoader

inputs_raw = np.stack([X_latitude, X_longitude], axis=1)
targets_raw = Y_elevation.reshape(-1,1) # turns a vector into a column matrix, same as doing Y_elevation[:, None]

# SPLITTING IN DIFFERENT SIZES
N = len(inputs_raw)     # 50K
n_train = int(0.7*N) # 35K
n_val = int(0.15*N)  # 7.5K
n_test = int(0.15*N) # 7.5K

# SPLITTING INDEXES (random but reproductible)
train_idx, val_idx, test_idx = torch.utils.data.random_split(
    np.arange(N),
    [n_train, n_val, n_test],
    generator=torch.Generator().manual_seed(42) # using a manual seed means that we can reproduce the results: we get the same pseudo-random values at each run
)

# BUILDING TRAIN, VAL & TEST ARRAYS BY THE RANDOM GENERATED INDEXES
X_train, Y_train = inputs_raw[train_idx], targets_raw[train_idx]
X_val,   Y_val   = inputs_raw[val_idx],   targets_raw[val_idx]
X_test,  Y_test  = inputs_raw[test_idx],  targets_raw[test_idx]
```

### 3. Normalizing the inputs and outputs between [0, 1]

Network training tends to struggle when the input data contain large numbers. That's why we will normalize the inputs.

```python
# GETTING INPUTS / OUTPUTS MIN MAX FROM TRAINING SET FOR NORMALIZATION
x_min_train, x_max_train = X_train.min(axis=0), X_train.max(axis=0)
y_min_train, y_max_train = Y_train.min(axis=0), Y_train.max(axis=0)

def normalize_inputs(X):
    return (X - x_min_train) / (x_max_train - x_min_train)

def unnormalize_inputs(X):
    return X  *(x_max_train - x_min_train) + x_min_train

def normalize_outputs(Y):
    return (Y - y_min_train) / (y_max_train - y_min_train)

def unnormalize_outputs(Y):
    return Y  * (y_max_train - y_min_train) + y_min_train

# NORMALIZATING TRAIN, VAL, TEST INPUTS BETWEEN [0,1]
X_train_norm = normalize_inputs(X_train)
X_val_norm   = normalize_inputs(X_val)
X_test_norm  = normalize_inputs(X_test)

# NORMALIZATING TRAIN, VAL, TEST OUTPUTS BETWEEN [0,1]
Y_train_norm = normalize_outputs(Y_train)
Y_val_norm   = normalize_outputs(Y_val)
Y_test_norm  = normalize_outputs(Y_test)

# CHECKING STANDARD DEVIATION
print("TRAIN elevation STD: ", Y_train_norm.std())
print("VAL elevation STD: ", Y_val_norm.std())
print("TEST elevation STD: ", Y_test_norm.std())
```

### 4. Converting the datasets into Pytorch tensors

Tensors are multidimensional arrays containing elements of the same data type. Tensors are the core class used for deep learning operations. Automatic differentiation is implemented on Pytorch tensors, so any transformation applied to the inputs by the network or the loss can be accounted for when evaluating the gradient.
